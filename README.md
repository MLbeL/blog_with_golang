# Blog With Golang

## About The Project

Blog With Golang is a backend blog application written in Go.

The main goal of this project is to practice building REST APIs, 
working with PostgreSQL, containerizing applications with Docker, 
and designing a scalable backend architecture.

This repository is a pet project created to improve backend development skills and explore architectural approaches in Go.

---

## Tech Stack

- Go (`net/http`)
- PostgreSQL
- Docker & Docker Compose
- Apache Kafka (planned)
- REST API

---

## Project Structure
```
cmd/ - Application entry point
internal/ - Core business logic
pkg/ - Reusable packages
configs/ - Application configuration (planned)
```

## Learning Goals

This project focuses on:

- Designing RESTful APIs in Go
- Working with relational databases (PostgreSQL)
- Using Docker for local development and containerized environments
- Preparing the foundation for event-driven architecture with Kafka
